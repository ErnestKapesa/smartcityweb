# Smart City Solar Website

A professional, multi-page website for Smart City Solar - a leading provider of advanced solar energy solutions in Africa.

## Features

### Multi-Page Structure
- **Home** - Hero section with company overview and key features
- **Products** - Detailed product pages for N-Type modules, TOPCON modules, and ESS
- **Technology** - In-depth look at solar technology and innovations
- **About** - Company information, mission, vision, and values
- **Contact** - Contact form and company information

### Professional Design
- Modern, clean interface with gradient accents
- Professional Font Awesome icons (no emoticons)
- Responsive design for all devices
- Advanced CSS animations and transitions
- AOS (Animate On Scroll) library integration

### Advanced Features
- Smooth scrolling navigation
- Animated statistics counters
- Parallax effects
- Scroll progress indicator
- Back-to-top button
- Mobile-friendly hamburger menu
- Form validation and submission handling
- Lazy loading for images
- Ripple effects on buttons
- Hover animations and transitions

### Technologies Used
- HTML5
- CSS3 (with CSS Grid and Flexbox)
- JavaScript (ES6+)
- Font Awesome 6.4.0
- AOS (Animate On Scroll) 2.3.1

## Setup Instructions

1. Ensure all files are in the correct directory structure
2. Open `index.html` in a web browser
3. Navigate between pages using the navigation menu

## Customization

### Colors
Edit the CSS variables in `css/styles.css`:
```css
:root {
    --primary-color: #0066cc;
    --secondary-color: #00a3e0;
    --accent-color: #ffa500;
    /* ... */
}
```

### Content
Update the HTML files to modify text, images, and structure.

### Animations
Adjust animation settings in `js/script.js`:
```javascript
AOS.init({
    duration: 800,
    easing: 'ease-in-out',
    once: true,
    offset: 100
});
```

## Performance Optimization

- Minify CSS and JavaScript for production
- Optimize images (compress and use appropriate formats)
- Enable browser caching
- Use a CDN for external libraries
- Consider lazy loading for images
